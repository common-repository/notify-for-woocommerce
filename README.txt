== Description ==

Notify for WooCommerce adds the ability to send SMS notifications for every status of the order. Every status trigger
can be enabled or disabled and customized. For the moment we have only Romanian (RO) customers. If you want this to your country, let us know at hello@notify.biz.

== Installation ==

1. Create an account on notify.biz
2. Install the plugin
3. Configure the API key, SIM and Device ID from the Settings page
